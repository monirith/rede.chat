# Gemini Live API — Supported Languages

97 languages supported by `gemini-3.1-flash-live-preview` for both audio input and audio output. The model can switch between languages naturally mid-conversation.

Source: https://ai.google.dev/gemini-api/docs/live-guide

## Notes

- **No `gsw` code** — Swiss German is not a separate language to Gemini; treated as a German variant. Mundart dialect coaching is achieved via system prompts on top of the German model.
- **Romansh (`rm`)** — Switzerland's fourth official language, ~40k speakers. Worth supporting for the Swiss positioning.
- **Hebrew uses `iw`** — Google retains the deprecated ISO code (modern code is `he`).
- **Filipino uses `fil`** — three-letter exception in an otherwise two-letter list.
