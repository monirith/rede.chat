# Feature: Free Talk Mode

Open-ended conversation with no scenario frame. User picks dialect, talks about anything.

## Behaviour
- No scenario prompt injected — just the dialect system prompt
- Gemini acts as a friendly native speaker open to any topic
- Same 15-minute cap
- Same post-session feedback
- Good for intermediate users who want natural conversation vs guided scenarios

## Difference from Scenario
- No role, no tasks, no goal hints
- Gemini follows the user's lead
- Still corrects mistakes, still explains dialect differences

## V2
- User can set a topic ("let's talk about football / the weather / work")
- Unlimited free talk sessions for premium tier
