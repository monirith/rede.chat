# Backlog: Gamification (XP, Leaderboards, Tiers)

Speak.com equivalent: XP system, weekly leaderboard, tier promotions, streak freeze.

## Why it's backlog
- Our user (professional, Zurich-area, well-paid) is not Duolingo's target
- Motivation is extrinsic: job, social life, integration — not points
- Leaderboard requires user base large enough to be meaningful
- Adds complexity for zero launch benefit

## If we add it later
- Streak freeze (pay credits to protect streak) — natural monetization hook
- Dialect badges ("Züridütsch Insider", "Berndütsch Beginner")
- Keep it minimal — achievement milestones, not competitive leaderboard
